How our service is different

  • The data warehouse allows its users to access the data and create customized visualizations without fear of overriding the data.

  • The Statistics and Data Analytics Team is available for questions or calls if there are many questions a user has.

  • As other institutions request certain visualizations or fields that can be beneficial to all institutions, they can be added to your standard analytics.

  • Aside from only being able to visualize the data, every visual created can be exported as an Excel file or a picture. Additionally, the entire “tab” or page of visuals can be downloaded as a pdf.

  • You can add “read-only” users who will be able to look at the visuals you create, but will not be able to change them.

  • The user “authors” are able to create analyses and dashboards and share them with read only users.
